## 四、赛道体系

三个主要赛道，选择最适合的方案：

| 赛道 | 说明 |
|------|------|
| **结构化流程编排 (Structured Process Orchestration)** | 使用 Maestro BPMN 设计固定流程的自动化 |
| **智能体案例管理 (Agentic Case Management)** | 处理动态、异常密集型工作流，由Agent自主决策 |
| **智能体测试 (Agentic Testing)** | AI驱动的自动化测试方案 |

## 五、技术栈要求

| 要求 | 说明 |
|------|------|
| **UiPath Coding Agents** | 新平台能力，支持 Claude Code / Codex / Gemini CLI 创建AI Agent |
| **UiPath Automation Cloud** | 编排与治理层，项目必须运行于此 |
| **外部框架** | LangChain, CrewAI, AutoGen 等欢迎使用 |
| **UiPath Labs** | 提供沙箱环境，含 Agentic + AI 单元 |
| **开源** | 用自然语言桥接原型→生产软件 |
